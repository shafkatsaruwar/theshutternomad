# The Shutter Nomad

Photography portfolio and booking site for The Shutter Nomad.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The site is a Vite/React single-page app. Vercel uses `vercel.json` to build
the project and route all client-side paths back to `index.html`.

## Hosting

This project is intended to deploy from GitHub to Vercel. It has no Base44
runtime dependency.
